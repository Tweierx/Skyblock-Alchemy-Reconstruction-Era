# Skyblock-Alchemy-Reconstruction-Era
Alchemy Skyblock: A 1.20.1 Minecraft modpack where you start with just an oak tree and a dirt block, surviving by decomposing resources into elements, crafting tools via recipe overhauls, and expanding islands through alchemical progression.
